# 🎬 YouTube Shorts 자동화 시스템

AI를 활용한 유튜브 쇼츠 자동 생성 및 업로드 시스템

## ✨ 기능

- 🤖 **AI 스크립트 생성** (Google Gemini)
- 🎙️ **자동 음성 생성** (Edge TTS)
- 🎬 **배경 영상 자동 수집** (Pexels)
- ✂️ **영상 자동 합성** (MoviePy)
- ☁️ **Google Drive 저장**
- 📺 **YouTube 자동 업로드**
- 📱 **Telegram 알림**
- 📊 **Google Sheets 대시보드**
